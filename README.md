### 사용 기술:
*  인공지능(AI)
* 파이썬(Python)
* 테서렉트(tesseract)
* 오픈CV(OpenCV)
* 데이터베이스(MySQL)
* 파이큐티(PyQt5)

### 프로그램의 흐름
   1. 카메라에 번호판을 대고 등록 버튼을 누른다.
   2. 번호가 등록되어있으면 경고문이 나오고 없으면 데이터가 저장이되고 DB에도 insert 된다.
   3. 카메라에 번호판을 대고정산 버튼을 누른다.
   4. 번호가 없으면 경고문이 나오고 있으면 DB에 update 된다.
   5. 등록된 차메뉴 버튼을 누르면 현재 입력되있는 번호판이 출력된다.
